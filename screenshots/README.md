Suggested capture guidance:

- Use the `Mission Control` tab with the cockpit theme active
- Keep the sidebar telemetry rail visible in at least one shot
- Show the session list and operator notes/checklist panel together
- Capture the session detail drawer if a session is selected
- Capture the health radar, config diff, and activity timeline if they fit in one frame or across multiple frames
- Include one frame for the `Refresh` action
- Include one frame for the plugin `Rescan` action
- Include one frame for the safe config snapshot panel so redaction is visible

Suggested demo order:

1. Open Hermes Dashboard and switch to `Mission Control`
2. Pause on the cockpit overview and point out the sidebar telemetry rail
3. Open a session to show the detail drawer
4. Highlight the health radar, config diff, and activity timeline
5. Click `Refresh` and then `Rescan Plugins`
6. Finish with the safe config snapshot and operator notes/checklist
